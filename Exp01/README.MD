# 数据输入
此脚本内定义了一个numpy三维数组data

```python
data[0] = ([[144,18],
            [136,44],
            [129,15],
            [150,10],
            [157,30],
            [151,51]])

data[1] = ([[324,16],
            [316,42],
            [309,10],
            [330,16],
            [337,30],
            [331,50]])

data[2] = ([[195,36],
            [188,00],
            [180,35],
            [202,6],
            [208,45],
            [203,11]])

data[3] = ([[375,34],
            [368,0],
            [360,30],
            [382,4],
            [388,45],
            [383,6]])

```
其中第一维（axis=0）

```data[0]```为 $\theta_1$

```data[1]```为 $\theta_2$

```data[2]```为 $\theta^{'}_{1}$

```data[3]```为 $\theta^{'}_{2}$

第二维（axis=1）中的六个值为六次测量得到的数据

第三维（axis=2）中的两个值为数据的度和分，在后续计算中程序会将其转换为小数格式

### 注意
部分数据测量过程中越过了刻度盘零点，就需要在输入数据的时候手动加上360°

例如```data[3,1,1]```中，原始数据为8°，减去```data[1,1,1]```的316°显然不在52°附近

换句话说，需要保证```data[3]-data[1]``` ```data[2]-data[0]```的值在52°附近
